# 파이썬 퀵 스타트 - 업무에 바로 쓰는 파이썬

### 공지 사항
- sources 폴더에는 각 장별로 책의 소스가 있습니다.
- 변경사항 폴더에는 소스 변경 사항 및 의존성 변경사항에 대한 텍스트 파일이 있습니다.
	- 이 내용들은 sources 폴더의 각 장에도 포함되어 있습니다.
- 오탈자 폴더에는 교재 오탈자가 등록되어 있습니다.
- 11장에서 pptx 모듈 사용시 모듈 의존성 관련된 부분이 변경 되었습니다. 상세 문서는 11장의 문서를 참고해 주세요
- 12장에서 크롤링의 특성상 크롤링을 하고자 하는 웹 페이지가 변경되면, 크롤링 소스를 변경해 주어야 합니다.  웹 페이지의 변경 사항을 반영하여 변경된 소스 코드를 올려드립니다.
